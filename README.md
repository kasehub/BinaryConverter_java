# Binary Converter in Java

A simple Java program that converts a positive decimal integer into its binary equivalent using arrays and loops.

## Features
- Accepts user input
- Converts decimal numbers to binary
- Uses repeated division by 2
- Stores binary digits in an array
- Prints the binary number in correct order

## Concepts Used
- Arrays
- While loops
- For loops
- Modulus operator (%)
- Integer division
- Scanner class for user input

## How It Works
The program:
1. Takes a decimal number as input
2. Divides the number repeatedly by 2
3. Stores the remainders in an array
4. Prints the remainders in reverse order to form the binary number
